# Web

인터넷은 웹의 핵심적인 기술로 컴퓨터들이 서로 통신 가능한 거대한 네트워크입니다.\
컴퓨터는 인터넷 서비스 제공 업체 (Internet Service Provider, ISP) 에서 관리하는 라우터에 연결되어 있으며, 이를 통해 인터넷에 연결됩니다.

1. 인터넷의 작동 원리
   * DNS (Domain Name System) 탐색 과정
2. TCP vs UDP
   * 3 Way-Handshake & 4 Way-Handshake
3. HTTP vs HTTPS
   * 공개키 vs 개인키
4. HTTP 구성
5. HTTP 그외
   * Cookie vs Session
   * URI vs URL vs URN
   * Forward vs Redirect
